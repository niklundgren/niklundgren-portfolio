# Portfolio

This directory contains the React + Vite source for the portfolio site. The app uses `react-router-dom` for page routing, keeps shared UI in `src/components`, and organizes route-level content under `src/pages`.

## Development

Install dependencies and run the local dev server:

```bash
npm install
npm run dev
```

Other common commands:

```bash
npm run build
npm run preview
npm run lint
```

## Top-Level Directory Guide

- `src/`: application source code.
  `main.jsx` mounts the app, `App.jsx` defines routes, `components/` holds shared UI, `pages/` contains route components, and `data/` stores static content such as `scholar.json`.
- `public/`: static assets copied through Vite without bundling transformations.
- `dist/`: production build output generated by `npm run build`.
- `index.html`: Vite HTML entry point used to bootstrap the React app.
- `package.json`: project metadata, scripts, and runtime dependencies.
- `package-lock.json`: npm lockfile for reproducible installs.
- `vite.config.js`: Vite configuration for local development and production builds.
- `eslint.config.js`: ESLint configuration for the React codebase.
- `.gitignore`: ignore rules for generated files and local-only artifacts.

## Site Structure

- `src/components/Navigation.jsx`: persistent site navigation with nested project and skills links.
- `src/pages/Home.jsx`, `Projects.jsx`, `Skills.jsx`, and `Contact.jsx`: primary top-level routes.
- `src/pages/projects/`: detail pages for project-specific views such as the phonon viewer, CAD designs, and teaching resources.
